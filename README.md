# data-analytics-portfolio
My Excel, Power BI, SQL and Python projects portfolio.


This repository showcases a collection of end-to-end data analytics projects developed using **Excel, Power BI, SQL, and Python**. The projects focus on transforming raw datasets into actionable insights through data cleaning, exploration, visualization, statistical analysis, and business-oriented reporting.

The portfolio is divided into two main case studies:

## Consumer Electronics Sales Analysis (Excel & Power BI)

This project analyzes a consumer electronics dataset containing information on products, sales transactions, customers, stores, and currencies.

Key analyses and visualizations include:

* Identifying the top-performing sales channels and delivery locations
* Analyzing customer purchasing behavior across different age groups
* Comparing sales performance by product category, brand, and size
* Examining trends in product demand and average order values
* Converting and standardizing sales values in USD
* Monitoring currency fluctuations over time
* Building interactive dashboards to visualize sales performance, customer segmentation, and geographic distribution

The project demonstrates how business intelligence tools can be used to support data-driven decision-making in the retail and consumer electronics sector.

## Pharma Store Data Analysis (SQL & Python)

This project focuses on a fictional pharmacy retail dataset and explores customer behavior, product performance, and sales trends using SQL and Python.

Key tasks include:

* Writing complex SQL queries to answer business questions
* Identifying top-selling products, channels, and cities
* Analyzing customer purchase patterns and order histories
* Calculating average order values and customer-level metrics
* Segmenting customers by age groups and demographics
* Investigating sales distributions and detecting outliers across brands
* Performing statistical hypothesis testing to compare sales channels
* Creating visualizations to explore pricing patterns and sales performance

The project highlights practical applications of data analysis techniques, including data manipulation, aggregation, statistical testing, and exploratory data analysis.

## Tools and Technologies

* **Excel** – Data preparation, pivot tables, and dashboards
* **Power BI** – Interactive reports and business intelligence dashboards
* **SQL (Oracle)** – Data extraction, transformation, and querying
* **Python** – Data cleaning, analysis, visualization, and statistical testing
* **Pandas, Matplotlib, SciPy** – Data processing and analytics libraries

This portfolio reflects a hands-on approach to solving real-world business problems and demonstrates proficiency in transforming data into meaningful insights.
